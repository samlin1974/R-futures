
策略：
  	入場：連續三天紅K 且收盤價一天比一天高 隔天開盤價進場買股票
	出場：入場後，第一個收盤價超過MA05均價 隔天開盤價賣股票(也可改MA20)

其他條件：
	時間區間：2010至今(程式碼當中可改)
	進出場成本：買/賣價 3%

注意：執行時請確保網路暢通
	
檔案說明：
	S3.R：前述策略個股回測(2330.TW)與提供單次交易入出場圖形 供驗證(內有個股代碼 可自行調整)
	S31.R: 前述策略測試所有列表的930檔股票，以驗證該策略適合某股票股性
	stockcode.csv:前述待測股票代碼 可自行調整(台股請以.TW結尾)
	resulttbl.csv：前述S31.R執行完900之股票後的結果 
		欄位說明：
			meanRet #"平均報酬率
                  	tradeNums #交易次數
                  	winRatio #勝率
                  	sdRet #報酬率標準差
                  	maxRet #最大報酬率
                  	minRet #最小報酬率           
                  	avgHoldDays #平均持有日數
                  Eg. 代碼2548.TW股票按前述策略 再給定的時間內 
			共有七次交易 勝率為71% 平均報酬 1.75% 持有天數平均13天
處理效能：(MacBook Pro / i5 CPU/8G Ram)
	S3.R 約10秒
	S31.R 約9分鐘
